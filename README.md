# MOST self-hosted

Этот репозиторий содержит скрипты и шаблоны для развертывания и поддержки сервиса МОСТ.

## Быстрый запуск

Запускайте команды от root (или через sudo).

### 1) Получить исходники

```
cd /opt
git clone https://github.com/snikket-im/snikket-selfhosted snikket
```

### 2) Настроить сервер

```
cd snikket
./scripts/init.sh
```

### 3) Запустить сервис

```
./scripts/start.sh
```

Проверьте, что страница входа МОСТ открывается по адресу `http://<ваш-домен>/`
(после выпуска сертификатов произойдет редирект на HTTPS).

Скрипт `./scripts/start.sh` дождется статуса `healthy` для `snikket_server`
и выведет итоговый статус готовности в консоль.

### 4) Создать первую учетную запись администратора

Важно: создавайте приглашения только после того, как `snikket_server` перешел в состояние `healthy`.

```
./scripts/new-invite.sh --admin --group default
```

Откройте ссылку приглашения в браузере.

## Кастомный брендинг МОСТ

В репозитории добавлена папка `branding/` с логотипом МОСТ и шаблонами страниц прокси/портала.
При запуске через `docker-compose.base.yml` эти файлы автоматически монтируются в контейнеры,
поэтому на странице входа и служебных страницах отображается логотип МОСТ без упоминаний Snikket.
